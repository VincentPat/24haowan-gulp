# 24haowan-gulp
24好玩工作流

### Description
24好玩前端工作流，主要内容是自动构建和刷新浏览器预览
### Dependencies
- browser-sync
- del
- gulp
- gulp-autoprefixer
- gulp-cached
- gulp-file-include
- gulp-notify
- gulp-rename
- gulp-rev
- gulp-rev-collector
- gulp-sass
- gulp-uglif

### Get Started
##### Clone
```
$ git clone git@github.com:GuangZhouShanyouGame/24haowan-gulp.git
```
##### Intallation
```
$ npm install
```
### Usage
##### Build Your Project
Attention: Default task will clean the dist directory first.
```
$ gulp
```
##### Watch File Change
It will open a new page automatically and refresh when the files you are watching change.
```
$ gulp watch
```
